-saya mengelempokkan elemen elemen menggunakan div supaya mudah di styling dan mengatur layoutnya. Pada tugas ini terdapat div container(utama), div profil dan div social

background-position: center untuk posisi gambar di tengah, background-size: cover untuk gambar menutupi seluruh halaman, background-repeat: no-repeat agar gambar tidak berulang,
height dan width untuk mengatur ukuran area background (ketinggian dan kelebaran background)
-Di tugas ini juga menerapkan Flexbox, saya menggunakan: Display: flex untuk mengaktifkan mode flexbox untuk mengatur layout, justify-content: center 
supaya posisi elemen ke tengah secara horizontal, align-items: center supaya posisi elemen ke tengah secara vertikal. Jadi konten berada di tengah halaman
-selain itu ada beberapa kode lainnya seperti border-radius untuk membuat sudut menjadi bulat seperti yang diterapkan pada image profil.
-kode gap untuk mengatur jarak, color untuk mengatur warna, font-size untuk mengatur ukuran teks, position untuk mengatur posisi elemen(relative, absolute)
